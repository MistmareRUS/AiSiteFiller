Инструкции для нейронки

0. Регулярно ломаются наши с тобой беседы. Здесь будут вводные инструкции, чтобы мы были на одной волне.
1. Изучить данный репозиторий, рассказать что видишь, вывести мне краткое описание, как ты понимаешь работу этого приложения.
2. В твоем редакторе есть проблема с передачей ссылок, некоторые сегменты пропадают. Решение - сслки в коде ты собираешь через конкатенацию строк. А те, что присылаешь мне, склеиваешь через "пробел плюс пробел"
3. Не нужно самодеятельности при расширении нашего веера рассылки. Уже есть реализованные сервисы, делаем по аналогии.
4. Придерживаемся чистой архитектуры, не сусорим в проекте.
